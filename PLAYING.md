# Playing ReMoM

ReMoM is a source-code reconstruction of **Master of Magic** (Simtex, 1994-95).
You need the original game data files to play.

## Quick Start

1. Obtain the original **Master of Magic v1.31** (DOS) game files.
   You need every `.LBX` file and `CONFIG.MOM`.

2. Copy those files into this folder (the same folder as `ReMoMber`).

3. Run `ReMoMber` (or `ReMoMber.exe` on Windows).

## Troubleshooting

- **"Cannot open MAGIC.LBX"** or similar — the game data files are missing
  or not in the same directory as the executable. Make sure all `.LBX` files
  and `CONFIG.MOM` are in this folder.

- **Black screen on launch** — try resizing or alt-tabbing. Some systems
  need a moment for the SDL window to initialize.
